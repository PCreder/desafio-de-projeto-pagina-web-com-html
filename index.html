O que é HTML?

HTML (Linguagem de Marcação de Hipertexto) é um codigo que utilizamos para criar uma Pagina Web.

Ferramentas Utilizadas

Para escrever paginas HTML precisamos apenas de um editor de textos, como o bloco de notas, por exemplo.

Porém, ao utilizarmos um editor de código próprio para escrever em HTML, nosso trabalho como desenvolvedor se torna mais fácil e ágil. Um exemplo de editor é o Visual Studio Code.

Inspecionar

O inspetor de elementos é uma ferramenta utilizada no próprio browser.

Permite inspecionar os elementos na página. É uma ótima forma de estudar como uma página foi criada.

Para usar é só apertar a tecla f12 no teclado, ou botão direito em qualquer lugar da página web, e escolher a opção inspecionar.

Estrutura

O documento HTML é criado através de tags(marcaçoes no documento) consiste principalmente em tres partes:

  1. HTML

    2. É a tag inicial do documento, indica que ele é um documento HTML. Dentro dessa teg temos mais duas outras:

  2. Head

    3. Tudo que estiver entre essa tag será carregado antes da página ser exibida para o usuário.

  3. Body

    4. Tudo que estiver entre essa tag irá ser exibido para o usuário.


Inicio

Tags

São marcações inseridas no documento HTML que ajudam a dar forma ao texto passando instruções ao navegador.

Exemplos de Algumas:

  • h1 até h6: São tags de título. Sendo a h1 de maior tamanho e importância, diminuindo até a h6, menor tamanho e importância.

  • p: É uma tag de parágrafo, para usar em textos em geral.

  • mark: Torna o texto grifado por uma marca texto.

  • small: Diminui a letra em algum trecho específico.

  • i: Deixa o texto em itálico.

  • u: Deixa o texto em sublinhado.

  • strong: Deixa o texto em negrito.

  • ol: Cria listas ordenadas.

  • ul: Cria uma lista não ordenada, com marcadores em vez de numeradores.

  • li: Colocado dentro da tag ol ou ul, para identificar os itens da lista.

  • a: É uma ancora, cria ligação com outra página.

  • hr: Cria uma linha de separação de ponta a ponta na página

  • sub: Para colocar caracteres acima da linha em que o resto do texto está escrito, normalmente ele também fica em uma fonte menor.

  • sup: Para colocar caracteres abaixo da linha em que o resto do texto está escrito, normalmente ele também fica em uma fonte menor.

  • blockquote: Cria uma citação.

  • input: Cria um campo para digitação. O que irá ser aceito nesse campo será escolhido através de um atributo

  • img: É uma tag para colocar uma imagem, ela precisa do atributo src com o caminho da imagem.

Atributos

Propriedades que uma tag pode ter. São palavras especiais usadas dentro da tag para controlar o comportamento do elemento.

Alguns exemplos são o ID e o Class. Ambos servem para identificar o elemento, sendo que cada ID so pode ser usado em um elemento, e o Class pode ser repetido quantas vezes forem necessárias.

Também tem o atributo Style, ele serve para adicionar comandos de estilo diretamente no documento HTML.
